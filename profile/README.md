# Wordic

**Wordic**은 데이터베이스 컬럼명을 한글로 작성하고 이를 일관성 있는 영어 스네이크 표현식으로 변환해주는 혁신적인 도구입니다.

## 소개

데이터베이스 컬럼명을 일관된 형식으로 변환하면 데이터 관리와 분석 작업이 훨씬 간편해집니다. **Wordic**을 통해 데이터베이스 스키마를 표준화하고, 데이터 작업의 효율성을 극대화하세요.

## 주요 기능 및 장점

- 한글 컬럼명을 영어 스네이크 표현식으로 변환
- 데이터베이스 스키마의 일관성 유지
- 데이터 분석 및 쿼리 작업을 용이하게 함
- 사용자 정의 규칙 및 옵션을 통한 유연성 제공
- 직관적이고 사용하기 쉬운 인터페이스
